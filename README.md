This backend exposes a single /chat endpoint that accepts `{ "message": "..." }` and replies with Ms. Lisa's response generated by OpenAI.

## Quick start (local)

1. Copy `.env.example` to `.env` and set `OPENAI_API_KEY`.
2. Install dependencies:
   ```
   npm install
   ```
3. Start:
   ```
   npm start
   ```
4. Test:
   ```
   curl -X POST http://localhost:5000/chat -H "Content-Type: application/json" -d '{"message":"Hello Lisa"}'
   ```

## Deploying to Render

- Create a new Web Service on Render.
- Upload this repository (or connect Git).
- Set the `OPENAI_API_KEY` environment variable in Render's dashboard.
- Build Command: `npm install`
- Start Command: `npm start`
- After deploy, use the service URL and point your frontend `index.html` fetch to `https://your-render-url/chat`.

## Notes & Safety
- This server uses the OpenAI API. Do not commit your API key to source control.
- The system prompt enforces Lisa's persona. Edit `systemPrompt` in `server.js` to tweak behaviour.
